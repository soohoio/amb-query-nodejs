# amb-query-nodejs

### 1. Authenticated AWS on your local machine

```
export AWS_ACCESS_KEY_ID="{input here}"
export AWS_SECRET_ACCESS_KEY="{input here}"
```

### 2. use node 14 version (using nvm dependency)
```
nvm use 14
```

### 3. install dependencies
```
npm install
```
